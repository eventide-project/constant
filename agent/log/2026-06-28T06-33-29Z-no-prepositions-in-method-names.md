# Rule: never favor prepositions in method names — a preposition refers to the argument, and arguments are inherent to methods, so no name-word need announce them (the parameter does); favoring them is a bias toward the mean. Rubric must be all-or-none, so none. Scopes to method/initializer names only — prose keeps load-bearing prepositions ("mediates for", "sent to"). → rule `…06-33-29Z-no-prepositions-in-method-names.md`
