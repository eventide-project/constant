# `full_name` is two words, not one (`fullname`). "Full name" is an adjective (`full`) modifying a noun (`name`) — it has not lexicalized into a closed compound the way `filename`/`username` have, so English keeps the space and snake_case keeps the underscore. It also preserves the kinship with the method it lengthens: `#name` → `#full_name` (vs. `fullname`, which reads as a separate opaque token). No code change — confirms the existing name.
