# Code style: don't inline a method call as an argument — bind its result to an explaining local first, then pass the local (`raw_constant = namespace.const_get(...); new(raw_constant)`, not `new(namespace.const_get(...))`). The intermediate is a meaningful domain value; naming it documents intent and gives an inspection point.
