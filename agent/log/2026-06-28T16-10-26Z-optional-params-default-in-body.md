# Code style: optional parameters default to `nil` in the signature; their real defaults are assigned in the body via `||=` (`namespace=nil, inherit: nil` ‚Üí `namespace ||= Object; inherit ||= false`), not inline in the parameter list. More robust (an explicit `nil` argument is coerced to the default too, unlike an inline default which only applies when the arg is omitted) and keeps all defaulting visible in one place in the body.
