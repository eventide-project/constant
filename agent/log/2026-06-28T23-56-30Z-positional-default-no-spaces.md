# Style: a positional-argument default is written tight — no spaces around `=` (`namespace=Object`, not `namespace = Object`). Distinct from a keyword default (`inherit: false`, keeps the colon-space) and an ordinary assignment (`x = y`, keeps its spaces); the `=` binds the parameter to its default as one unit. The library code already followed it (`def self.build(name_or_module, namespace_name_or_module=nil, inherit: nil)`); codified here and extended to signatures quoted in prose — fixed the design doc's `build`/`defined?` signatures and the plan's `namespace=Object` reference. → rule `…23-56-30Z-positional-default-no-spaces.md`
