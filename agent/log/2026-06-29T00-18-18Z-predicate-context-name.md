# Naming: a predicate method's test `context` is named "`<Name>` Predicate" — the method name without the `?`, plus "Predicate" (`defined?` → `context "Defined Predicate"`, not "Defined?"). Names what the method *is* (a predicate) literally, and keeps the `?` out of a context label where it reads as interrogative prose.
