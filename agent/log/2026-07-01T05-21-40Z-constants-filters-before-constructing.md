# Constant::Module#constants filters the raw Ruby constants by module-ness BEFORE constructing Constant instances (filter_map: const_get, then build only the kept ones), rather than wrapping every inner via #get and discarding the literals — avoids allocating Constant::Literal objects that would be thrown away in the default case. The cost is #constants doing its own const_get + type dispatch (not routed through #get, which wraps unconditionally and would defeat the efficiency); no shared helper extracted (kept as-is)
