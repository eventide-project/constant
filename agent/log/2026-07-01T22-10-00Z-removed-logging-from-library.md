# Removed logging from the library — the developer decided not to add logging. Logging was only ever scaffolded, never invoked: `Constant::Log` (tagging messages `:constant`) was defined but called by nothing. Removed: `lib/constant/log.rb` and its two requires (`require "log"`, `require "constant/log"`); the README `## Log Tags` section (which documented log messages that were never actually written); and the design doc's Out-of-Scope "Logging for the `Constant` class … revisit later" note. Also dropped the now-obsolete **`evt-log` gemspec runtime dependency** — `evt-log` was a direct dependency of this gem alone (nothing else in the dependency tree needed it), pulled in solely for `Constant::Log`. Re-installed the packages without it (`bundle install --standalone default development`), pruning `evt-log` plus ~9 transitive-only packages (evt-telemetry, evt-clock, evt-dependency, terminal_colors, evt-mimic, evt-subst_attr, evt-reflect, evt-record_invocation, evt-invocation) from the installed bundle and its load path. `Gemfile.lock` and `gems/` are git-ignored, so the only committed dependency change is the gemspec line. Verified: 75 tests
