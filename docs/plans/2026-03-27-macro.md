# Constant::Import Macro Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add macro interface to `Constant::Import` so that `include Constant::Import` gives a module or class the `__import_constant` and `import` class-level macros.

**Architecture:** A `self.included(base)` hook on `Constant::Import` extends the receiver with `Constant::Import::Macro`. `Macro` defines `__import_constant` as the concrete method (delegating to `Import.call` via `**kwargs`) and aliases it as `import`. The existing `Import.call` API is untouched.

**Tech Stack:** Ruby 4. TestBench for tests — uses `context`, `test`, `assert`, `refute` blocks. Run a single file with `ruby <path>`, run all tests with `ruby test/automated.rb`.

        end
      end
    end

      end
    end
  end
end

**Step 3: Add `self.included` and `Macro` to `lib/constant/import.rb`**

Insert after `Error = Class.new(RuntimeError)`:

```ruby
def self.included(base)
  base.extend(Macro)
end

module Macro
  def __import_constant(source_constant, **kwargs)
    Import.(source_constant, self, **kwargs)
  end

  alias import __import_constant
end
```

`self.call` is unchanged.

**Step 4: Run the test to verify it passes**

Expected: PASS

require_relative "../../automated_init"

context "Import Constant" do
  context "Macro" do
    context "Alias" do
      receiver_constant = Controls::Constant.example(name: "Receiver")

      source_constant = Controls::Constant.example(name: "Source") do
        const :SomeInnerConstant
      end

      alias_constant_name = :SomeAliasConstant

      receiver_constant.include(Constant::Import)
      receiver_constant.import(source_constant, alias: alias_constant_name)

      alias_constant = receiver_constant.const_get(alias_constant_name, false)

      comment "Source Constant: #{source_constant.inspect}"
      comment "Receiver Constant: #{receiver_constant.inspect}"
      comment "Alias Constant Name: #{alias_constant_name.inspect}"
      comment "Alias Constant: #{alias_constant.inspect}"

      context "Alias constant is defined in receiver" do
        defined = receiver_constant.const_defined?(alias_constant_name, false)

        detail "Defined: #{defined.inspect}"

        test do
          assert(defined)
        end
      end

      context "Inner constants are nested in alias" do
        defined = alias_constant.const_defined?(:SomeInnerConstant, false)

        detail "Defined: #{defined.inspect}"

        test do
          assert(defined)
        end
      end

      context "Inner constants are not at receiver root" do
        defined = receiver_constant.const_defined?(:SomeInnerConstant, false)

        detail "Defined: #{defined.inspect}"

        test do
          refute(defined)
        end
      end
    end
  end
end
